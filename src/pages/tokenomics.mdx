# Tokenomics

|              |               |
| ------------ | ------------- |
| Name         | SLDX          |
| Ticker       | $SLDX         |
| Supply       | 1,000,000,000 |
| Type         | Utility       |
| Availability | Solana        |
|              |               |

SLDX is the native token of the Slide App platform, with a hard capped supply of 1B units, set to exhibit deflationary properties over time. SLDX is designed as a utility token that services many key functions within the network. SLDX is currently available on the following blockchains:
| Blockchain | Contract|
| ------------ | ------------- |
| Solana | DEVNET |

## Token Functionality

- Use SLDX to buy gSLDX and play games on the Slide App platform.
- Use SLDX to battle against others.

## gSLDX

gSLDX is a fictional token that can be used to play games on the Slide App platform. This token can be bought using SLDX and withdrawn to SLDX.

By ficitional token is meant that this token is not listed on a blockchain.

| gSLDX amount | Fixed price |
| ------------ | ----------- |
| 10,000       | $1          |

**How are SLDX and gSLDX related?**

1. **Buy gSLDX**:
   The gSLDX amount you want to buy is calculated as following: the SLDX amount you want to pay times 10,000 times the current SLDX price in dollars.
   ![buy-gSLDX](/images/gSLDX-formula.png)

2. **Withdraw gSLDX**:
   The SLDX amount you receive when withdrawing gSLDX is calculated as following:the amount of gSLDX you want to withdraw over 10,000 and over the current SLDX price in dollars.
   ![withdraw-gSLDX](/images/SLDX-formula.png)

**Example 1: You want to withdraw 85,500 gSLDX to SLDX.**

| Utility                         | Amount |
| ------------------------------- | ------ |
| gSLDX to withdraw               | 85,500 |
| current $SLDX price             | $2,50  |
| SLDX after withdraw (formula 2) | 3,42   |
| SLDX (in dollars)               | $8,55  |
